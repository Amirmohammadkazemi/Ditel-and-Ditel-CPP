# How to programming C++ book project
This is code of ***Grade Book*** project in how to programming c++ book.

This project writen on ***Clion IDE*** and need som config.

## Config
Maybe when you want to run executable file, you see tis error ***libstdc++-6.dll is missing*** or something like that.
For fix this error must go to setting -> Languages & Frameworks -> C/C++ -> Clangd
and add ```-static-libgcc``` and ```-static-libstdc++``` in Clang Errors and Warnings part.

## Run
For run this project you have 2 way:

1. Open Clion and create executable project and put the old files in place of the new files.
2. Open this project on Clion and run it.
